# 📄 [EnterLab] 요구사항 정의서 (PRD.md)

## 프로젝트 개요

> **AI 1:1 코칭 예약 및 자동 알림 웹서비스 (국내망 기반)**
> 한국 내 사용자를 대상으로, AI에 대한 이해가 부족한 초보자도 개인 맞춤형 코칭을 통해 AI 수익화 기술을 빠르게 익히고 실행할 수 있도록 돕는 **국내형 AI 코칭 예약 플랫폼**입니다.

---

## 핵심 타겟 (Persona)

* **자영업자 / 중소사업자**: AI 자동화, 문서 자동화에 관심 있는 사용자
* **2030 세대**: 진로 전환 및 부업으로 AI를 배우려는 사람
* **5060 세대**: AI 접근이 어려운 중장년층, 맞춤 코칭이 필요한 사용자

---

## 주요 기능 (MVP Scope)

1. **1:1 코칭 예약 폼**

   * 입력 항목: 이름, 연락처, 이메일, 희망 날짜/시간, 코칭 형태(출장/사무실), 선택 과목, 요청사항, 개인정보 동의
   * DB 저장 + 이메일 및 문자 자동 발송 (SendGrid + Naver Cloud SENS)

2. **자동 안내 시스템 (국내망 기반)**

   * 예약 완료 시:

     * 사용자에게 예약 완료 이메일 및 SMS 발송 (SENS API)
     * 관리자에게 신규 예약 알림 이메일 발송 (SendGrid)
   * 향후 결제 안내(계좌이체 정보 포함) 자동화 가능

3. **브랜딩 랜딩 페이지**

   * EnterLab 슬로건 및 한 줄 소개 반영

     * "고민은 끝, 수익을 실행하는 단 하나의 키."
     * "혼자서는 1년, EnterLab과는 단 4주 코스."
   * 신뢰감 있는 UI (jeongdami.vercel.app 스타일)

---

## 사용자 흐름 (User Flow)

1️⃣ 접속 → 2️⃣ 코칭 신청서 작성 → 3️⃣ 개인정보 동의 후 제출
→ 4️⃣ DB 저장 → 5️⃣ 이메일 및 SMS 자동 발송 → 6️⃣ 완료 안내 페이지 표시
→ 7️⃣ 고객 전화는 050 가상번호로 착신되어 내 번호 노출 방지

---

# 🤖 [EnterLab] AI 코딩 착수용 프롬프트 (Tasks.md)

```
너는 Ruby on Rails 8.0 (Hotwire 포함)의 수석 개발자야.
첨부한 [PRD]와 [TRD] 문서를 기반으로, 100% 국내망 환경에서 작동하도록 구현해.

아래 사양을 반드시 준수해야 해.
1️⃣ 예약 폼 구현 (이름, 연락처, 이메일, 날짜/시간, 코칭형태, 선택과목, 요청사항, 개인정보동의)
2️⃣ 제출 시 DB에 저장 후 이메일 및 문자 자동 발송
   - 이메일: SendGrid API 사용
   - 문자: Naver Cloud SENS API 사용
3️⃣ 전화 문의는 KT 050 가상번호로 연결되어 내 실제 번호를 숨김
4️⃣ ActionMailer + REST API 방식으로 비동기 처리
5️⃣ Tailwind 기반 UI와 Hotwire 폼 애니메이션 적용
6️⃣ jeongdami.vercel.app와 유사한 레이아웃 및 신뢰감 있는 디자인 구현
7️⃣ 개인정보 보호법을 준수해 DB에 고객정보를 암호화 저장

💡 중요: Twilio, 해외 API는 절대 사용하지 않는다.
```
