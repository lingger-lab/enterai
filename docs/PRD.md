# [Enter.ai] 요구사항 정의서 (PRD)

## 프로젝트 개요
> AI 1:1 코칭 예약 및 자동 알림 웹서비스  
AI에 대한 이해가 부족한 초보자도, 개인 맞춤형 코칭을 통해 AI 수익화 기술을 빠르게 익히고 실행할 수 있도록 돕는 “AI 기반 코칭 예약 플랫폼”

## 핵심 타겟 (Persona)
- 자영업자 / 중소사업자
- 진로 전환 및 부업을 고려하는 2030 세대
- AI 접근이 어려운 5060 세대

## 주요 기능 (MVP Scope)
1. 1:1 코칭 예약 폼
2. 자동 안내 시스템 (이메일 / 문자 발송)
3. 브랜딩 랜딩 페이지

## 사용자 흐름
1️⃣ 접속 → 2️⃣ 신청서 작성 → 3️⃣ 개인정보 동의 및 제출  
→ 4️⃣ DB 저장 → 5️⃣ 이메일 및 문자 자동발송 → 6️⃣ 완료 안내 페이지 표시
