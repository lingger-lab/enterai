## 2. 디자인 시스템 일관성

### 2.1 타이포그래피

폰트 사용 빈도 (index.html.erb):
| 클래스 | 빈도 | 용도 |
|--------|------|------|
| text-sm | 65 | 보조 텍스트 |
| text-xs | 42 | 캡션/메타 |
| text-xl | 27 | 카드 제목 |
| text-lg | 27 | 본문 강조 |
| text-2xl | 25 | 섹션 제목 |
| text-base | 17 | 본문 |
| text-4xl | 14 | 헤드라인 |
| text-3xl | 9 | 서브 헤드라인 |
| text-6xl | 1 | 히어로 |

→ ✅ **9단계 위계 일관**. text-xs(12px) 비중 큰 점은 검토 가능 (이미 시니어 모드 도입).

### 2.2 그림자 (Elevation)

| 클래스 | 빈도 | 의도 |
|--------|------|------|
| shadow-sm | 18 | 카드 평소 |
| shadow-md | 9 | 약한 강조 |
| shadow-lg | 8 | 강한 강조 |
| shadow-xl | 7 | CTA 호버 |
| shadow-2xl | 1 | 최대 강조 |

→ ✅ **5단계 위계 의도적**. 일관성 있음.

### 2.3 색상

- 메인: indigo-600 (CTA, 강조)
- 페르소나별:
  - 직장인: indigo
  - 시니어: amber
  - 소상공인: emerald
- 중성: gray-700/600/500
- 위험: red-500/600

→ ✅ **컬러 시스템 명확**. Tailwind config의 primary 정의 활용 여지 (현재는 indigo 직접 사용).

### 2.4 둥근 모서리

- `rounded-2xl` (24px) — 카드/버튼 큰 요소
- `rounded-xl` (16px) — 폼 컨트롤
- `rounded-lg` (8px) — 작은 버튼/태그
- `rounded-full` — 배지/아이콘

→ ✅ **4단계 일관**.

### 2.5 간격 (Spacing)

- 섹션 패딩: `py-10 sm:py-24`
- 카드 내부: `p-6 sm:p-8`
- 작은 카드: `p-4 sm:p-5`
- 버튼: `px-5 py-2` 또는 `px-8 py-4` (CTA)

→ ✅ **일관성 있음**. 한국 디자인 표준.

## 3. 사용자 흐름 (User Flow) 평가

### 3.1 신규 사용자 (비회원)

```
검색/카톡 링크 → 홈
  → 스크롤 (히어로 → 고민 → 커리큘럼 → 코치 → 슬라이드 → 가격 → 후기 → CTA)
  → "무료 상담 신청" 클릭
  → 10단계 폼 작성 (이름→연락처→이메일→일시→코칭형태→과목→요청→패키지→동의→확인)
  → 예약 완료 페이지 (토큰 URL)
  → SMS/이메일/카톡 알림 수신
```

| 평가 | 점수 |
|------|------|
| 검색→홈 | ⭐⭐⭐⭐⭐ |

→ **전체 8.6점**. 시니어 친화적.

## 13. 새로 발견된 PERSPECTIVE (사용자 미언급)

### 13.1 카카오톡 인앱 브라우저 (한국 특수)

- 한국 사용자 30%+ 카톡 링크로 사이트 진입
- 카톡 인앱 브라우저 = 자체 WebView
- 일부 JS API 미지원 (geolocation, FileReader 등)
- 우리 사이트 영향: 거의 없을 듯 (단순 HTML/CSS/Stimulus)
- **테스트 필요**: 실 디바이스에서 카톡으로 링크 공유 후 모든 기능 동작 확인

### 13.2 시니어 사용자 특수 고려

| 항목 | 평가 |
|------|------|
| 큰 글씨 모드 | ✅ 푸터 버튼 |
| 클릭 가능 영역 | ✅ py-3/4 (충분) |
| 콘텐츠 길이 | ⚠️ 홈 904줄 — 시니어에게 부담 가능 |
| 글자 색상 대비 | ✅ 개선됨 |
| 음성 안내 | ❌ (선택) |

### 13.3 페이지 로딩 순서 (LCP 최적화)

- 히어로 텍스트 = LCP (Largest Contentful Paint)
- 현재 히어로 = 단순 텍스트 → ✅ 빠름

- **개선**: 히어로는 JS 없이도 즉시 표시되도록 (현재 OK이지만 검증 필요)
